Was versteht man unter Data Analytics?¶

  • Der Begriff Data Analytics umfasst alle Aktivitten zur Gewinnung von Informationen aus Daten.
  • Das sind das Sammeln, Speichern, Organisieren der Daten sowie die Analyse der Daten und die Prsentation der Ergebnisse. Link
  • Manchmal werden die Begriffe Data Analytics und Data Analysis (auf deutsch Datenanalyse) als synonym verwendet. Es gibt jedoch einen kleinen Unterschied.
  • Data Analytics ist ein umfassenderes Konzept. Das bezieht sich auf den Prozess der Verwendung von Daten um Erkenntnisse zu gewinnen und fundierte Entscheidungen zu treffen. Link
  • Die Datenanalyse ist ein fundamentaler Teil der Data Analytics. Sie bezieht sich auf das Bereinigen, Umwandeln und Modellieren von Daten, um ntzliche Informationen zu finden.

Die vier Methoden der Data Analytics¶

Die Data Analytics wird zustlich in Aktivitten bzw Methoden unterteilt:

  • Descriptive Analytics - Was ist passiert?
  • Diagnostic Analytics - Warum ist das passiert?
  • Predictive Analytics - Was wird in der Zukunft passieren?
  • Prescriptive Analytics - Was sollte getan werden, damit dies passiert?

Data Analysis vs Data Science¶

Data Analysis:

  • Der Datenanalyst befasst sich mit der Sammlung und Aufbereitung von Daten sowie der Untersuchung von Trends, Beziehungen und einfachen Prognosemodellen.
  • Zu den Instrumenten der Datenanalyse gehren hauptschlich deskriptive Statistiken, Regressionen (linear und logit), Zeitreihenkonometrie und Clusteranalyse.
  • Andere Skills sind sehr wichtig: Prsentationen halten, Berichte schreiben.

Data Science:

  • Data Science, auf deutsch Datenwissenschaft, konzentriert sich hauptschlich auf die Predictive Analytics.
  • In Data Science werden fortgeschrittene Tools genutz, wie Machine Learning, Deep Learning und natural language processing (zur Analyse von Texten).
  • Mehr Kenntnisse in Mathematik und Statistik sind erforderlich.
  • Der Data Scientist kmmert sich um das Modell und die Aktualisierung der Modellparameter, wenn neue Daten zur Verfgung stehen.
  • Data Science $\neq$ Data Engineering.
