Was ist Python?¶

  • Python ist eine Programmiersprache, die in verschiedenen Bereichen eingesetzt wird (Softwareentwicklung, Webentwicklung, Datenanalyse).
  • Entwickelt von Guido van Rossum, der von der Community genannt „lebenslanger wohlwollender Diktator“, bis er im Jahr 2018 von seinem „Amt“ zurücktrat. 📘Link
  • Python ist eine so genannte „höhere“ Programmiersprache, d. h. sie ist weiter von der Maschinenebene entfernt, aber die Syntax ist für Menschen leichter zu verstehen.
  • Ein „Interpreter“ wandelt Code von einem für Menschen verständlichen Code in Maschinencode um. 📘Link
  • Zahlreiche Bibliotheken erweitern die Grundfunktionalität von Python in verschiedenen Bereichen. Python-Pakete können zum Beispiel von PyPI (Python Package Index) oder über Anaconda heruntergeladen werden.
  • Laut den auf GitHub gesammelten Daten ist Python eine der am häufigsten genutzten Programmiersprachen der Welt. 📘Link

R

  • ist eine Programmiersprache von Statistikern für Statistik entwickelt
  • keine fortgeschrittenen Programmierkenntnisse erforderlich
  • mit R und den zahlreichen R-Paketen lassen sich wunderschöne Diagramme und Dokumente erstellen (Bücher, Berichte) und in verschiedenen Formaten (pdf, HTML)
  • hauptsächlich in der akademischen Forschung/Wissenschaft verwendet

Python

  • ist eine allgemeine Programmiersprache (general-purpose programming language) genutz in verschiedensten Anwendungsbereichen
  • leicht zu lernen (ähnlich wie R)
  • hat sich schnell zu einer der meistgenutzten Programmiersprachen der Welt entwickelt
  • hat sich stark in der Wirtschaft etabliert, weniger in der Wissenschaft
  • Python-Kenntnisse werden von Unternehmen in vielen Bereichen (Finanzen, Beratung usw) häufig gefragt

Mit welchen Werkzeugen werden wir arbeiten?¶

Anaconda¶

  • Anaconda ist eine Python-Distribution.
  • Mit Anaconda installiert man Python sowie zahlreiche Python-Pakete für Datenanalyse und Data Science.
  • Anaconda erleichtert die Paketmanagement (besser nicht Anaconda und pip mischen!).
  • conda ist das Paketverwaltung-System.
  • Befehle werden über die Kommandozeile eingegeben (Anaconda Powershell Prompt).
  • Einige Editionen der Software sind kostenpflichtig aber es gibt auch eine kostenlose Version.

Jupyter Notebook¶

  • Jupyter Notebook ist eine Software zur Entwicklung und Verteilung von datengesteuerten Dokumenten (unterstützt verschiedenen Programmiersprachen).
  • Es ist ein Open-Source-Software von Project Jupyter entwickelt.
  • Es ist webbasiert, d. h. es funktioniert über den Browser.
  • Ermöglicht die Erstellung professioneller Dokumente und Präsentationen im HTML-Format.
  • Dank Markdown kann man Text und mathematische Formeln mit Code leicht kombinieren.
  • Es wird automatisch mit Anaconda installiert.
  • Die neue Version ist JupyterLab.

Spyder¶

  • Spyder ist eine Open-Source-Entwicklungsumgebung, auf Englisch „integrated development environment“ (kurz IDE).
  • Es wurde speziell für Python und Data Science entwickelt.
  • Einfach und intuitiv zu bedienen, ideal für Anfänger!
  • Es ist Teil der Anaconda-Distribution.

Git¶

  • Git ist ein Open-Source-Tool zur Versionskontrolle.
  • Git verfolgt Änderungen an Dokumenten und ermöglicht auf ältere Versionen des Dokuments zurückzugreifen.
  • Ermöglicht die Zusammenarbeit mehrerer Entwickler an einem Projekt durch die Einrichtung von „Zweigen“ (branches) des Projekts.
  • Macht die Projektentwicklung nachvollziehbar und transparent.

GitHub¶

  • Ist GitHub dasselbe wie Git? Nein!
  • Git ist ein Versionskontrollsystem. GitHub ist eine Webschnittstelle für Git-Projekte.
  • Git funktioniert auch ohne GitHub. GitHub ohne Git macht wenig Sinn.
  • GitHub bietet zahlreiche zusätzliche Funktionen für Zusammenarbeit und Projektentwicklung.
  • GitHub ist ein kommerzielles Produkt, aber die Grundfunktionen sind kostenlos (es gibt andere ähnliche Open-Source-Tools, z. B. GitLab).

Was versteht man unter Data Analytics?¶

  • Der Begriff Data Analytics umfasst alle Aktivitäten zur Gewinnung von Informationen aus Daten.
  • Das sind das Sammeln, Speichern, Organisieren der Daten sowie die Analyse der Daten und die Präsentation der Ergebnisse. 📘Link
  • Manchmal werden die Begriffe Data Analytics und Data Analysis (auf deutsch Datenanalyse) als synonym verwendet. Es gibt jedoch einen kleinen Unterschied.
  • Data Analytics ist ein umfassenderes Konzept. Das bezieht sich auf den Prozess der Verwendung von Daten um Erkenntnisse zu gewinnen und fundierte Entscheidungen zu treffen. 📘Link
  • Die Datenanalyse ist ein fundamentaler Teil der Data Analytics. Sie bezieht sich auf das Bereinigen, Umwandeln und Modellieren von Daten, um nützliche Informationen zu finden.

Die vier Methoden der Data Analytics¶

Die Data Analytics wird zusätlich in Aktivitäten bzw Methoden unterteilt:

  • Descriptive Analytics - Was ist passiert?
  • Diagnostic Analytics - Warum ist das passiert?
  • Predictive Analytics - Was wird in der Zukunft passieren?
  • Prescriptive Analytics - Was sollte getan werden, damit dies passiert?

Data Analysis vs Data Science¶

Data Analysis:

  • Der Datenanalyst befasst sich mit der Sammlung und Aufbereitung von Daten sowie der Untersuchung von Trends, Beziehungen und einfachen Prognosemodellen.
  • Zu den Instrumenten der Datenanalyse gehören hauptsächlich deskriptive Statistiken, Regressionen (linear und logit), Zeitreihenökonometrie und Clusteranalyse.
  • Andere Skills sind sehr wichtig: Präsentationen halten, Berichte schreiben.

Data Science:

  • Data Science, auf deutsch Datenwissenschaft, konzentriert sich hauptsächlich auf die Predictive Analytics.
  • In Data Science werden fortgeschrittene Tools genutz, wie Machine Learning, Deep Learning und natural language processing (zur Analyse von Texten).
  • Mehr Kenntnisse in Mathematik und Statistik sind erforderlich.
  • Der Data Scientist kümmert sich um das Modell und die Aktualisierung der Modellparameter, wenn neue Daten zur Verfügung stehen.
  • Data Science $\neq$ Data Engineering.
